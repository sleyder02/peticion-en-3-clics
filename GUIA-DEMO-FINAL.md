# Guía de Demo Final


## Flujo recomendado

1. Abrir la plataforma.
2. Crear una causa comunitaria.
3. Registrar un apoyo.
4. Ver contador.
5. Descargar PDF.
6. Explicar advertencia jurídica.
7. Explicar n8n/Telegram.
8. Mostrar README final.
9. Mostrar checklists.
10. Cerrar con reflexión.

## Mensaje de presentación

Petición en 3 Clics demuestra cómo una comunidad puede organizar una solicitud, registrar apoyos y generar evidencia documental con tecnología sencilla y responsable.
