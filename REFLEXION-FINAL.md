# Reflexión Final
```

Durante el programa aprendí que la tecnología puede servir para organizar ideas, comunicar necesidades, crear soluciones y apoyar procesos ciudadanos.

El reto final me permitió ver que una causa comunitaria puede convertirse en evidencia organizada cuando se combina comunicación clara, cuidado jurídico y desarrollo tecnológico.

También aprendí que no todo se trata de programar. Una solución responsable debe cuidar datos personales, explicar sus límites y evitar exponer información innecesaria.

Este proyecto demuestra que incluso una herramienta sencilla puede tener valor cuando ayuda a una comunidad a expresarse mejor.
